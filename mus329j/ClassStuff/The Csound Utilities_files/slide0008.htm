Syntax of convolve
lain = input (excitation) signal
lifilcod = convolve.#  or “filename”
lichannel = optional channel specifier
lNote: output will be delayed by length of impulse file
lDelay the input “dry” signal by this amount if you wish to align it with the “wet” signal from convolve
ar1[, ar2[, ar3[, ar4]]] convolve ain, ifilcod, ichannel
