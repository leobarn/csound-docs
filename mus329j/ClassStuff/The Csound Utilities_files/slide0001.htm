The Csound Utilities
lCsound provides a number of utilities that can either be executed as stand-alone programs or invoked via the command line option -U
lA number of Csound’s most interesting unit generators require data generated by one of these utilities
lIn most cases, the utilities operate on standard (integer) audio files (WAV or AIFF)
lSome utilities require the user to set critical parameters (such as analysis window size, hop size, etc) that directly affect the results (see the Csound manual)
