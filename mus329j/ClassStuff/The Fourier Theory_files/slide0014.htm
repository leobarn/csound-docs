Syntax of pvanal
csound -U pvanal [flags] infilename outfilename
infilename – audio file to be analyzed
outfilename – output file name (usually pvoc.nnn)
Flags:
-s rate – sampling rate of the audio input file.
-c channel – channel number sought. Default is all channels.
-b begin – beginning time (in seconds). Default is 0.0
-d duration – duration (in seconds). Default is to the end of the file.
