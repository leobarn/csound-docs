More on pvanal
Additional Flags:
-n frmsiz – STFT frame size, the number of samples in each analysis frame.
  Must be a power of two, in the range 16 to 16384
-w windfact – Window overlap factor. This controls the number of Fourier
transform frames per second. The default value is 4.
Do not use this flag with -h.
-h hopsize – STFT frame offset. Converse of above, specifying the increment
in samples between successive frames of analysis.
Do not use with -w.
