Macintosh Install Instructions

  1. Download the correct csound5 for your platform (10.3.x or 10.4.x) from http://csound.sourceforge.net/
  2. Install the Csound5.mpkg (this installs all of the various libraries and basic GUIs.
  3. Download MacCsound (http://csounds.com/matt/MacCsound1.3b1.dmg), which is a front-end that has capabilities that are somewhat similar to WinXound Pro.

If you are using MacCsound, the basic environment variables can be set in the options. If you are old school and working on the command line or using one of the basic frontends installed with csound5 package, you will need to do a little extra work to establish your environment variables. Instructions are in the readme that comes with Csound5.
