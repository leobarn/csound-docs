Instructions for Downloading and Installing DOS Csound on a Windows PC:

The current PC/DOS Csound executables are available at the University of Bath, England. The files are compressed in the .zip format. Windows XP will automatically unzip zipped files, earlier versions of windows may require the WinZip application, available from www.winzip.com

Instructions:

 If you need WinZip, download the free evaluation version at http://www.winzip.com/ and follow the instructions to install it on your PC.

 Create a directory (folder) on your PC called \CSOUND. This is where you will download and install the Csound executables. Then create another directory called \SOUNDS. This will be where the audio files created and/or processed by Csound will be stored. In most cases, these directories will be on the C: disk, which will be assumed for these instructions.

 Link to the Bath site at http://www.cs.bath.ac.uk/pub/dream/newest. Click the filename "csound_con423f8.zip" to begin the download. Be sure to save the file to your C:\CSOUND directory.

 Open the csound_new.zip file (possibly renamed csound_n.zip when it was saved) and extract the files it contains into the C:\CSOUND directory.

  Edit your system variables as follows. You can edit them by right-clicking on "My Computer," selecting Properties, then click Environment Variables under the "Advanced" tab. You must be an administrator to change the PATH variable. Add the C:\CSOUND directory to the PATH statement and the following variables:

SFDIR=C:\SOUNDS

SSDIR=C:\SOUNDS

SADIR=C:\CSOUND

  If you make System variables, csound will see them for every user on your computer; User variables only affect your account. Csound is now ready to run. Note that you will probably want to use one of the available Windows front end applications to actually launch it, unless you are quite comfortable in the DOS environment. You may want to start by installing WinXound
