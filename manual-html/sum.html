sum

sum — Sums any number of a-rate signals.

Description

Sums any number of a-rate signals.

Syntax

ares sum asig1 [, asig2] [, asig3] [...]

Performance

asig1, asig2, ... -- a-rate signals to be summed (mixed or added).
