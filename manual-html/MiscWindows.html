Appendix E. Window Functions

Windowing functions are used for analysis, and as waveform envelopes, particularly in granular synthesis. Window functions are built in to some opcodes, but others require a function table to generate the window. GEN20 is used for this purpose. The diagram of each window below, is accompanied by the f statement used to generate the it.

Hamming. 

Example E.1. Hamming window function statement 

f81   0   8192   20   1   1


Hamming Window Function.

Hanning. 

Example E.2. Hanning window function statement 

f82   0   8192   20   2   1


Hanning Window Function

Bartlett. 

Example E.3. Bartlett window function statement 

f83   0   8192   20   3   1


Bartlett Window Function

Blackman. 

Example E.4. Blackman window function statement 

f84   0   8192   20   4   1


Blackman Window Function

Blackman-Harris. 

Example E.5. Blackman-Harris window function statement 

f85   0   8192   20   5   1


Blackman-Harris Window Function

Gaussian. 

Example E.6. Gaussian window function statement 

f86   0   8192   20   6   1


Gaussian Window Function

Rectangle. 

Example E.7. Rectangle window function statement 

f88   0   8192   -20   8   .1


Note: Vertical scale is exaggerated in this diagram.

Rectangle Window Function

Sync. 

Example E.8. Sync window function statement

Sync Window Function
