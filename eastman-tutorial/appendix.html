APPENDIX 1

Csound score files

This appendix contains standard Csound score files for all of the orchestra/score pair examples within this tutorial. These Csound score files are intended for use by non-ECMC users who do not have access to Alexander Brinkman's score11 Csound score file pre-processor.

Note: Several of these examples employ Csound unit generator soundin or diskin and require appropriately named input soundfiles.
