Even though Csound only provides a limited number of stock GEN
subroutines, it includes one (GEN 1) which can read in an
external file. Hence, one can use a standalone program
to generate functions that are not standard (with the limitation
that files need to be written in a format understandable for GEN
1). In fact, as this example shows, one can use csound itself to
create them. 

The run will produce 513 values for the specified function.
(Risset 1969: #513)
