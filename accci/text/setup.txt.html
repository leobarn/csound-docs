Setup information

The ready-to-run text versions of the instruments in this catalog
vary for different operating systems.  Unix, Mac and Dos/Windows
all use different new-line formats in texts.  To run orchestras,
you have two options: save the html file as text or visit the
Amsterdam website at http://mars.let.uva.nl/pub/gather/accci and
download a version of the files for your operating system. 

The soundfiles needed for some of the instrument designs are 
included on this CDROM.  You have to copy the subdirectory "SfLib"
into your dedicated Csound soundfile directory on your hard drive.
The files occupy about 1MB disk space.  The subdirectory on the CDROM
where you find these soundfiles is /Accci/Sf/Sflib.

Alternatively you can use the links below to grab the files.
  • 88_01_1.TAB
  • 88_01_2.TAB
  • Santur1.aiff
  • snap.aiff
  • Speech1.aiff
Happy computing!
